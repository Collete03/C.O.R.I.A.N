Codebase Genius - Frontend

This directory contains the Streamlit frontend for Codebase Genius.

Description

A modern, responsive UI that provides an interface to the Codebase Genius API (running from the root main.py).

app.py: The main Streamlit application file.

Features

Modern, responsive UI

AI-enhanced documentation (via backend)

Real-time statistics (coming soon)

Download in Markdown and HTML formats

Generation history (loaded from the API)

Setup & Run

This application is not run directly from this folder. It is run by the main Streamlit command from the root of the project.

Activate Environment:

Make sure you are in the project's root directory and have activated your environment.

Run Streamlit:

From the root directory, run:

streamlit run frontend/app.py


The app will open in your browser at http://localhost:8501.
